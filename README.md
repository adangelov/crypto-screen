# Crypto Screen

Crypto Screen е инструмент за следене на пазарната структура на криптовалутни активи с помощта на интерактивни свещни графики и гъвкава решетка от карти.

## Локална разработка

Следните стъпки описват как да стартирате проекта локално:

1. Инсталирайте зависимостите:
   ```bash
   npm install
   ```
2. Стартирайте среда за разработка с hot-reload:
   ```bash
   npm run dev
   ```
   По подразбиране приложението е достъпно на [http://localhost:5173](http://localhost:5173).

## Генериране на production билд

За да създадете оптимизиран production билд изпълнете:
```bash
npm run build
```
Артефактите ще бъдат генерирани в директорията `dist/`.

## Преглед на production билда локално

След като сте изпълнили `npm run build`, може да прегледате резултата чрез вградения preview сървър:
```bash
npm run preview
```
Сървърът по подразбиране слуша на порт `4173`.

## Проверка на TypeScript типовете

Проектът използва TypeScript за статична проверка. За да валидирате типовете изпълнете:
```bash
npm run lint
```
